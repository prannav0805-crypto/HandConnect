# 🌌 HandConnect – Neon Aura AR

An immersive **web-based AR experience** that transforms your hands into glowing energy controllers using real-time tracking, physics, and sound.

Wave your hands… and the screen responds like it’s alive ⚡

---

## ✨ Features

* 🖐️ **Real-Time Hand Tracking**

  * Powered by MediaPipe Hands
* 🤏 **Gesture Recognition**

  * Pinch → Shockwave + zap sound
  * Open / Fist → Dynamic UI feedback
* 🌈 **Multiple Visual Themes**

  * Rainbow, Cyberpunk, Lava, Ocean, Galaxy
* ⚡ **Particle & Physics Engine**

  * Sparks, ripples, motion trails
* 🌧️ **Matrix Background Effect**

  * Reacts to hand velocity
* 🔊 **Interactive Audio System**

  * Sound changes based on hand distance
* 🌐 **Cross-Hand Interactions**

  * Lightning arcs between fingers
  * Dynamic gradient connections
  * Rotating mandala visuals
* 📊 **Live HUD**

  * FPS, gestures, hand detection stats

---

## 🛠️ Tech Stack

* HTML5 Canvas

## ⚙️ Setup & Run

```bash

cd HandConnect
```

### 2. Run locally

Open directly:

```bash
open index.html
```

OR use a local server (recommended):

```bash
npx serve
```

## 🎮 How to Use

1. Click **"Enter Experience"**
2. Allow camera access
3. Show your hands to the webcam
4. Try gestures:

   * 🤏 Pinch → Shockwave effect
   * ✋ Open hand → Expands visuals
   * ✊ Fist → Contracts visuals
5. Switch themes from bottom panel
